## 📖 概要

XERO Gamertag Checker は、ランダムに生成したゲーマータグが Xbox で使用可能かどうかを自動でチェックするツールです。非同期処理により、複数のチェックを同時に実行できます。

---

## ✨ 機能

| 機能 | 説明 |
|------|------|
| 🔀 **ランダム生成** | 指定した長さのゲーマータグを自動生成 |
| ⚡ **高速チェック** | 非同期処理で最大200件の同時チェック |
| 🔢 **数字切り替え** | 英字のみ / 英字+数字 を選択可能 |
| 🔊 **サウンド通知** | 利用可能なタグが見つかったら音で通知 |
| 📝 **自動保存** | 見つかったタグを自動でファイルに保存 |
| 🔄 **自動リトライ** | 失敗時は最大3回まで自動再試行 |

---

## 🚀 使い方

### 実行方法

**EXE版（推奨）:**
```
XERO_TagChecker.exe をダブルクリック
```

**Python版:**
```bash
python check.py
```

### 設定項目

起動すると以下の設定を入力できます（Enterでデフォルト値）:

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| ゲーマータグの長さ | 生成するタグの文字数 | 3 |
| 数字を含める | `y` = 英字+数字 / `n` = 英字のみ | n |
| 同時チェック数 | 並列で実行する数 | 50 |

### 出力例

```
チェック中 → abc          ← 使用中のタグ（上書き表示）
✓ 利用可能 → xyz          ← 見つかった！（保存される）
✓ 利用可能 → qwe          ← 見つかった！（保存される）
```

## ⚙️ 技術仕様

| 項目 | 詳細 |
|------|------|
| 言語 | Python 3.14 |
| 非同期処理 | asyncio + aiohttp |
| ビルド | PyInstaller |

### 最適化設定

- **DNSキャッシュ**: 5分間キャッシュして高速化
- **Keep-Alive**: TCP接続を再利用
- **タイムアウト**: 接続5秒 / 総計10秒
- **リトライ**: 失敗時は0.1秒後に再試行（最大3回）

## 🔧 ビルド方法

自分でEXEをビルドする場合:

```bash

# ビルド
pyinstaller --onefile --console --icon=icon.ico --name "XERO_TagChecker" ^
  --hidden-import=aiohttp --collect-all aiohttp check.py
```
